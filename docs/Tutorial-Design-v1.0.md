# Voikerchat チュートリアル詳細設計 v1.0

**作成日**: 2026-06-19  
**バージョン**: v1.0  
**ステータス**: 確定  

## 概要

新規ユーザー向けチュートリアル（オンボーディング）の完全仕様。5ステップで診断テスト実施 → レベル判定 → シーン選択まで導く。

### 設計方針
- ステップ数: 5ステップ（最小化と説明の両立）
- UI主体: アニメーション付き（UX重視）
- スキップ機能: 診断テスト内でスキップ可、後からレベル変更可
- ターゲット: フィリピン人配偶者向け（日本語初級～中級者）

---

## Step 3: 診断テスト

| 項目 | 仕様 |
|------|------|
| **問数** | 3問固定 |
| **難度段階** | Q1: N4（初級）/ Q2: N3（中級）/ Q3: N2（中上級） |
| **形式** | 4択 + 「わかりません」（5番目の選択肢） |
| **スコア計算** | 正解 = 1点、わかりません/スキップ = 0点 |
| **判定** | 3点→Advanced / 1-2点→Intermediate / 0点→Beginner |
| **スキップ機能** | 各問にてスキップ可 → テスト中断 → Step 4へ遷移 |

## UI/UX グローバルガイドライン

### カラースキーム
- 背景: フィリピン青#0047AB → 日本赤#DC143C グラデーション
- テキスト（主）: 白#FFFFFF
- テキスト（補）: ライトグレー#CCCCCC
- ボタン（プライマリ）: ゴールド#FFD700
- ハイライト: 黄色グロー#FFFF00

### フォント
- 日本語: 丸ゴシック（フレンドリー）
- 英語: Roboto / Inter sans-serif
- サイズ: タイトル28px / 本文14px / 説明12px

### アニメーション
- フェードイン: 0.3秒
- スライドイン: 0.4秒
- スケール: 0.2秒
- グロー: 常時

### 音声・振動フィードバック
- ボタンタップ: クリック音（100ms）+ 振動（10ms）
- 画面遷移: シュー音
- 正解回答: チャイム音 + 振動（20ms）
