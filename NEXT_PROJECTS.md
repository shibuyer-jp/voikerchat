## 🎮 Turntable Puzzle

### コンセプト（確定版）
古代ボードゲーム「メヘン」をモチーフに、ターンテーブルの 16 ステップパズルを進行。
**各ステップで流れてくる音のパーツをタップして配置 → 4ステップごとにコンボ完成 → ビート/メロディーが爆発的に良くなる**

> 「プレイヤーが『楽曲を作っている』という錯覚を得て、ビルドアップの視覚的快感を味わう」

### ターゲット層（確定）
- **年齢**: 10代～50代（全年代）
- **性別**: 問わない
- **共通特性**: フェス・イベント・クラブに躊躇なく参加する「文化体験消費層」

---

## 🎯 コアゲームプレイ（確定版）

### 配置モード
**「流れてくるパーツをタップ」（音ゲー的）**
- 画面下（または左）からチップが流れてくる
- 判定ラインに到達したタイミングでタップ
- **正確なタイミング = 快感**（参考：LUMINES の「タイムライン」）

### 差別化ポイント
1. **「配置ルール」という制約** — パズル的判断が必要
2. **「4ステップごとのビルドアップ」** — 段階的な音の快感上昇
3. **「ジャンル別にルール異なる」** — 同じゲームなのに全く違う体験

### 最高の見どころ
**「ビルドアップの視覚的快感」**
- ステップ 1-4：基本ビート完成 → ゴールド色に光る ✨
- ステップ 5-8：ベース追加 → 青色が加わる ✨✨
- ステップ 9-12：メロディー追加 → 虹色に変わる ✨✨✨
- ステップ 13-16：フルドロップ → 画面爆発、プリズム光 ✨✨✨✨

## 🎮 House Mode 完全仕様（ステップ 1-16）

### チップ配置ルール全体

**【ステップ 1-4】基本ビート**
- ステップ 1: KICK のみ → 「ドン」
- ステップ 2: HAT のみ → 「タ」
- ステップ 3: SNARE のみ → 「タン」
- ステップ 4: HAT のみ → 「タ」
- **結果**: 「ドン・タ・タン・タ」

**【ステップ 5-8】ベース追加**
- ステップ 5: KICK のみ → 「ドン」
- ステップ 6: BASS のみ（新規）→ 「ズーン」
- ステップ 7: SNARE のみ → 「タン」
- ステップ 8: BASS のみ → 「ズーン」
- **結果**: 「ドン・ベース・タン・ベース」（前のステップの音に重ねて）

**【ステップ 9-12】メロディー追加**
- ステップ 9: KICK のみ → 「ドン」
- ステップ 10: CYMBAL のみ（新規）→ 「キラン」
- ステップ 11: SNARE のみ → 「タン」
- ステップ 12: CYMBAL のみ → 「キラン」
- **結果**: 「ドン・キラン・タン・キラン」（すべての層に重ねて）

**【ステップ 13-16】フルドロップ**
- ステップ 13: KICK または BASS（選択可）
- ステップ 14: CYMBAL または HAT（選択可）
- ステップ 15: SNARE または COWBELL（選択可）
- ステップ 16: すべてのチップが落ちてくる（最高潮）
- **結果**: 「完璧なハウストラック」（すべてが爆発的に重なる）
